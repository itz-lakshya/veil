# Anonymous Confession App

A full-stack anonymous confession platform built using:

- Node.js
- Express.js
- MongoDB
- Mongoose
- Vanilla JavaScript
- Tailwind CSS

## Features

- Post confessions
- Like/unlike confessions
- Topic filters
- Search confessions
- Real-time frontend rendering without reload

## Run Locally

```bash
npm install
node main.js
```

Then open:

http://localhost:3000
